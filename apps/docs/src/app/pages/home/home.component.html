A collection of modular libraries for use within Angular apps.

  • Control Errors open_in_new

    A modular directive that automatically displays error messages associated with form controls when validation errors occur.

  • Numeric Input open_in_new

    A directive designed to be used inside <mat-form-field> component from Angular Material and to format numeric input fields to adhere to locale-specific formatting rules, such as digit grouping (thousands separators), decimal mark usage (period or comma), and potentially other region-specific adjustments and to return a number value.
